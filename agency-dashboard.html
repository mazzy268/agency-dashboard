Setup guide
Everything you need to get started — all free, in order
Total cost to set up: £0. All tools below have free tiers that are sufficient to start. You only spend money when you earn money (Stripe takes 2.9% per transaction).
One-time setup — do these once
1
Get your Anthropic API key
This dashboard uses Claude AI to generate emails and copy. You need a free API key to enable the AI features.
→ console.anthropic.com → API Keys → Create Key
Once you have the key, paste it into the box below and save this file.
Saved!
Free tier available
2
Create a free Stripe account
You'll use Stripe to send payment links and collect money from clients. No monthly fee — they take 2.9% only when you get paid.
→ stripe.com → Create account (free)
After setup: Dashboard → Payment Links → Create Link → set amount → copy link → paste into invoice email.
Free ~10 mins
3
Create a free Carrd account
Carrd is where you'll actually build each client website. Free plan lets you publish sites. Pick a clean business template and reuse it for each client.
→ carrd.co → Sign up free → browse Business templates
Optional upgrade: Carrd Pro ($19/yr) lets you use custom domains — worth it once you have your first paying client.
Free to start ~5 mins
4
Set up a Google Form for client onboarding
When a lead replies to your email, you send them this form. Their answers go straight into the Copy Generator tab.
→ forms.google.com → Blank form → add these questions:
1. Business name & tagline
2. Services you offer
3. Phone number & email address
4. Brand colours (or colours you like)
5. Do you have a logo? (file upload)
6. Photos of your work (file upload)
7. What makes you different from competitors?
8. Google Maps listing URL (if you have one)
9. Domain name (if you have one)
10. Anything else you want on the site?
Free ~15 mins
5
Bookmark this dashboard
Save this file to your desktop or bookmark it in your browser. Open it every morning. It runs entirely in your browser — no internet required for the dashboard itself (only for AI features).
File location: wherever you saved agency-dashboard.html — double-click to open in Chrome.
2 mins
Your daily workflow (once set up)
A
Morning: find 10 leads (~20 mins)
Go to Find Leads tab → enter niche + city → follow the plan. Search Google Maps for businesses with no website. Note their name, type, city, and email in a Google Sheet. Add them to the Pipeline here.
B
Send cold emails from Gmail (~20 mins for 10 emails)
Go to Outreach Emails tab → fill in details → copy Email 1 → paste into Gmail → send. Do this for each lead. Set a reminder to send Email 2 in 3 days if no reply.
C
When someone replies → send them your Google Form
Reply saying: "Great! To get started, please fill in this quick form and I'll get working on your site: [your Google Form link]". Advance them to Qualified in the pipeline.
D
Build the site (~90–120 mins)
Copy their form answers → paste into Generate Copy tab → get full homepage copy. Open Carrd → duplicate your template → paste the copy in → add their logo/photos → publish. Advance them to Building, then Client Review.
E
Send preview → get approval → invoice (~10 mins)
Send them the Carrd preview link. When they approve, go to Invoice Email tab → generate email → create Stripe payment link → send both. Mark as paid when Stripe notifies you.
Hosting this dashboard online (optional — free)
To access this dashboard from any device (phone, other computer), host it free on GitHub Pages:

1. Create a free account at github.com
2. Click New repository → name it agency-dashboard → set to Public
3. Upload agency-dashboard.html → rename it to index.html
4. Go to Settings → Pages → Source: main branch → Save
5. Your live URL will be: https://yourusername.github.io/agency-dashboard

Note: your API key is stored in your browser's localStorage — it won't be visible in the public GitHub file.
